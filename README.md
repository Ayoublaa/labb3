Application Android : Formulaire d'Inscription et Transmission de Données
Ce projet est une application Android native illustrant la gestion des formulaires, l'interaction utilisateur et le passage de données entre deux activités (écrans).

📝 Description de l'exercice
L'objectif de cet exercice est de créer une interface de saisie d'informations personnelles et d'afficher ces données sur un second écran après validation.

Fonctionnalités principales :
Écran de saisie (Navigation) : Un formulaire complet comprenant des champs pour le nom, l'email, le téléphone, l'adresse et une liste déroulante (Spinner) pour la ville.

Validation et Envoi : Lors du clic sur le bouton "Envoyer", les données sont récupérées et transmises à l'activité suivante.
